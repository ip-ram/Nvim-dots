# Nvim-dots

### Установка
```bash
git clone https://github.com/ip-ram/Nvim-dots.git ~/.config/nvim/
```
### В neo vim
```
:Lazy sync
```
### debug
Leader b - toggle breakpoint,
F5 - continue (start),
F10 - step over,
F11 - step into,
F12 - step out,

### vim surround (обводит в выбраный символ выделеный текст)
S <символ>
